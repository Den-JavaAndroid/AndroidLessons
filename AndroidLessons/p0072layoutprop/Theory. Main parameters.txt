Layout width и Layout height - обычно указываются в dp или dip - Density-independent Pixels. Абстрактная ЕИ, позволяющая приложениям выглядеть одинаково на различных экранах и разрешениях.

sp - Scale-independent Pixels. То же, что и dp, только используется для размеров шрифта в View элементах/

Layout weight – вес. Свободное пространство распределяется между элементами пропорционально их weight-значениям.

Layout gravity - в FrameLayout используются для указания местоположения внутри (Все помещаемые в него элементы он по умолчанию помещает в левый верхний угол и никак их не выстраивает.)

Layout margin - это отступ. Он может быть со всех сторон сразу, либо только с необходимых сторон. 

Стили - Если у вас есть несколько элементов и вам надо, чтобы они выглядели одинаково, то вы просто создаете один стиль и применяете его к нужным элементам.
